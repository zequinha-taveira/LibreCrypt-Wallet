# LibreCrypt Wallet - Hardware

Este diretório contém os arquivos de design de hardware.

## Conteúdo (Planejado)

- `schematics/` - Esquemáticos do circuito
- `pcb/` - Design da PCB
- `bom/` - Bill of Materials
- `3d/` - Modelo 3D do enclosure

## Componentes Principais

| Componente | Modelo | Função |
|------------|--------|--------|
| MCU | RP2350 | Processador principal |
| Display | SSD1306 OLED | Interface visual |
| Botões | Tactile | Confirmação |
| Conector | USB-C | Comunicação + Alimentação |
